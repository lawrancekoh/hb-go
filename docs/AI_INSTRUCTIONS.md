# AI Coding Assistant Instructions

This file contains persistent instructions and rules for the AI coding assistant to follow during the development of this project.

## Workflow Rules

1.  **Documentation First**: Before proposing or performing any `git commit`, you **MUST** update all relevant documentation to reflect the changes.
    *   **Checklist**:
        *   `docs/TASK.md`: Update progress, mark completed items.
        *   `README.md`: Update if new prerequisites, commands, or features were added.
        *   Implementation Plans: Update if the plan changed during execution.
