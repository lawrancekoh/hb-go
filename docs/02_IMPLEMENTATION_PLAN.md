## Proposed Changes

### Project Initialization
#### Installation Commands
1.  **Create Project**: `npm create vite@latest . -- --template react`
    *   Initializes a new Vite project with React template in the current directory.
2.  **Install Dependencies**: `npm install`
    *   Installs standard React dependencies (`react`, `react-dom`).
    *   **Note**: Dependencies are installed into the local `node_modules/` folder, isolated from your system.
3.  **Install Router**: `npm install react-router-dom`
    *   Adds client-side routing capabilities.

### Styling Setup
#### Installation Commands
1.  **Install Tailwind**: `npm install -D tailwindcss postcss autoprefixer`
    *   Installs Tailwind CSS and its peer dependencies as dev dependencies.
2.  **Initialize Tailwind**: `npx tailwindcss init -p`
    *   Generates `tailwind.config.js` and `postcss.config.js`.
